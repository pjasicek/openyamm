# OpenYAMM Editor UI Kit

This package contains:
- `icons/` — starter SVG toolbar icons
- `openyamm_editor_ui_spec.md` — color, state, and usage rules
- `css_tokens.json` — machine-friendly token values for Codex

The icon pack is intentionally simple:
- 20×20 viewBox
- monochrome `currentColor`
- 1.7px stroke
- rounded caps and joins
- readable at 16px and 20px

Recommended default usage:
- toolbar icon box: 24×24 or 28×28
- icon glyph: 16×16 or 18×18
- icon + text for uncommon/destructive actions
- icon-only for repeated high-frequency tools
